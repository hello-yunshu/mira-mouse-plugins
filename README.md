# Mira Mouse Plugins

Declarative plugins for Mira. Packages contain JSON contracts, Fixtures, documentation, checksums, and signatures only; native code, scripts, web code, arbitrary WASM, remote resources, manufacturer binaries, and reverse-engineered source are forbidden.

- `mira.example-mock`: complete Fixture tutorial, `fixture-verified`.
- `mira.amaster`: read-only candidate based on local source evidence and offline Fixtures; not hardware-verified.
- `mira.logitech-hidpp`: unbundled research descriptor with an empty device whitelist.
- `mira.razer-viper`: unbundled research descriptor with an empty device whitelist.

No signed stable plugin Release exists yet. Do not distribute a preview as production-signed or claim model compatibility without the hardware evidence matrix.
